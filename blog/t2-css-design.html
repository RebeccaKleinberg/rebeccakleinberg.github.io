Classes vs. IDs

In the battle for proper selector usage, who would win?

Spoiler Alert: You will. Because you will use them correctly.

One of the most confusing things to learn when you are new to CSS to corrrecly use a class or id selector.

It is best to remember that classes can be used with many elements. You can crate one class with certain properties, and then apply that class to multiple elements. And elements can have multiple classes. It's a many to many situation

An id can only be associated with one HTML tag. There can be only one HTML tag associated with one id, and each id can have only one HTML tag. It's a one to one situation

For a class, you use a . in your CSS. For ID's, you use a a #
