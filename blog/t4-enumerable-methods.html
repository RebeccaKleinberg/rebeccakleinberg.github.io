It's Great to Enumerate

Traversing the Enumerable Module

The Enumerabable Module is one of the more esoteric concecepts in Ruby for the newb, because it has methods that work on a number of different classes, and you can even mix the Enumerable Module into your own classes when you create them, although the methods in the Enumerable Module are not all the same in any class, and simimlarly names methods do not work the same in eveyr class.

Consider the Enumerable#map method. This is a robust method that will allow you to iterate a block of code over an array, and produce an array that has the block of code effect each item. For instance, if you have an arraty of names, and you want them all to be to be capitalized, you can apply the .map method as follows:

names_arr = ["rebecca", "yanni", "james", "ramon", "ben"]
names_arr.map {|name| name.capitalize }
=> ["Rebecca", "Yanni", "James", "Ramon", "Ben"]

Each item in the array is capitalized, and the result in an array. The .map method iterates over the array and produces a result that is itself an array, without using a loop of any kind. This is a powerful option and just the start of what one can do the apply the Enumerable Module.
